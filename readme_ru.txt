Программа расчитывает области улавливания и траектории неферромагнитной частички для разных математических моделей.

Все физические модели находятся в пакете model.physicalModels.
На данный момент реализовано две модели:
1. тестовая.
2. модель, в которой частичка движется в параллельном магнитном поле.

Для получения готовой программы из исходника необходимо, чтобы была установлена JDK 8 или выше и maven.
Собственно, в корне проекта есть pom.xml, который указывает maven, как собирать проект.

После сборки проекта результат можно будет видеть в папке target: запускаемый jar и exe файлы. Собственно, любой из них
запускать можно.
